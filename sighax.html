What is SigHax?

Sighax is the brand new CFW that allows you to use CFW, but there is a HUGE upside: It can't be patched by Nintendo, since the firmware is possible by an oversight in the bootrom, and it could only be fixed physically.

What's the difference between Sighax and Boot9Strap?

Well, Boot9Strap is the user friendly version of CFW that exists purely because of Sighax. Sighax is the developer version that allows people to make CFW. Think of it as Root access on a mobile device.
